--ENGLISH--
ATTENTION!!! THIS IS A TEST / PROOF-OF-CONCEPT (POC) VERSION TO VERIFY THE POSSIBILITY OF ACCESSING World of Spectrum USING THE ESP32 WIFI CONNECTION.
Tested only on the LillyGo ESP32 VGA.
To update:

* Download the `firmware.upg` file from Releases.
* Copy the file to the SD Card.
* Use the option: `F1 -> Options -> Update -> Firmware`.

A new item will appear in the F1 menu: **Network Access**.
After that, enter your Wi-Fi credentials.
Set the proxy to: http://alternativebits.com/wos.php
Search for the file you want and enjoy!
USE AT YOUR OWN RISK!
